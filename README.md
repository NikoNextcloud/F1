# F1 Live API Hub 2026

Това е напълно автоматизирано уеб приложение за Формула 1 и Формула 2 (сезон 2026).

## Характеристики:
- **F1 / F2 превключвател**: отделни табове (Класиране, Календар, Отбори) за двете серии.
- **Всички отбори & Пилоти на живо (F1)**: Генерират се динамично на база текущите регистрирани конструктори чрез безплатното Jolpi (Jolpica/Ergast) API.
- **Автоматичен календар (F1)**: Чете състезанията директно от интернет и ги филтрира по дата.
- **Статус на сесията**: показва дали в момента тече тренировка/квалификация/спринт/състезание, или колко остава до следващата сесия — изчислено локално от разписанието (без нужда от платен live timing абонамент).
- **F2 данни**: класиране, състав по отбори и пълен календар за 2026. Няма безплатен public live API за F2, затова тези данни се поддържат ръчно във `f2-data.js` (виж коментарите в началото на файла как да се обновяват след всеки уикенд).
- **Гъвкав дизайн (Responsive)**: Оптимизиран за мобилни телефони и таблети.
- **Тъмен F1 стил**: Висококачествена тъмна тема.

## Стартиране:
Просто разархивирайте и отворете `index.html`. Готово за активиране чрез GitHub Pages!
