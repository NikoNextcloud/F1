# F1 Live API Hub 2026

Това е напълно автоматизирано уеб приложение за Формула 1 (сезон 2026), свързано с безплатното Jolpi API за данни в реално време.

## Характеристики:
- **Всички отбори & Пилоти на живо**: Генерират се динамично на база текущите регистрирани конструктори.
- **Автоматичен календар**: Чете състезанията директно от интернет и ги филтрира по дата.
- **Гъвкав дизайн (Responsive)**: Оптимизиран за мобилни телефони и таблети.
- **Тъмен F1 стил**: Висококачествена тъмна тема.

## Стартиране:
Просто разархивирайте и отворете `index.html`. Готово за активиране чрез GitHub Pages!
